# 🤖 AI 補助機能 使用料金・操作ガイド

## 📋 概要

この StoryBoardMaker の AI 補助機能は、OpenAI API を使用して以下の機能を提供します：

- **🔍 フレーム解析**: 前後のフレーム画像を解析してシチュエーションを理解
- **💬 AI 対話**: シチュエーションを基にカジュアルな会話で構図を相談
- **🎨 画像生成**: 対話内容を反映したフレーム画像の生成（予定）

---

## 💰 料金体系（2024 年 12 月現在）

### 🔍 フレーム解析（GPT-4o Vision）

- **モデル**: `gpt-4o`
- **入力**: $2.50 / 1M tokens
- **出力**: $10.00 / 1M tokens
- **画像**: $7.225 / 1K images

#### 実際の料金計算

```
1回の解析あたり：
- システムプロンプト: 約100 tokens
- 画像データ（1-3枚）: 約300-900 tokens相当
- AI応答: 約100-200 tokens

概算: 1回あたり 約0.5-1.5円
```

### 💬 AI 対話（GPT-4o Chat）

- **モデル**: `gpt-4o`
- **入力**: $2.50 / 1M tokens
- **出力**: $10.00 / 1M tokens

#### 実際の料金計算

```
1回の対話あたり：
- ユーザーメッセージ: 10-50 tokens
- システムプロンプト: 約150 tokens
- 対話履歴: 100-500 tokens
- AI応答: 50-150 tokens（短縮版）

概算: 1回あたり 約0.1-0.3円
```

### 🎨 画像生成（Replicate API）

- **FLUX Schnell**: $0.003 / 画像（約 0.5 円）
- **Stable Diffusion XL**: $0.002 / 画像（約 0.3 円）
- **生成時間**: 2-10 秒程度

#### 実際の料金計算

```
1回の画像生成あたり：
- FLUX Schnell（推奨）: 約0.5円
- SDXL（経済的）: 約0.3円

月50枚生成の場合: 約15-25円
```

## 📖 使い方ガイド

### 🚀 初期設定

#### 1. API 設定を開く

- 画面右側の AI 補助パネルで**⚙️ 設定**ボタンをクリック

#### 2. API キーを設定

```
対話AI設定:
- プロバイダー: OpenAI
- APIキー: sk-xxxxxxxxxx (OpenAIから取得)
- モデル: gpt-4o (推奨)

画像生成AI設定:
- プロバイダー: Replicate (推奨)
- APIキー: r8_xxxxxxxxxx (Replicateから取得)
- モデル: FLUX Schnell (推奨)
```

#### 3. 設定を保存

- **💾 保存して閉じる**をクリック

### 🎬 AI 補助機能の使用手順

#### Step 1: フレーム選択

1. 絵コンテの任意のフレームの**🤖 ボタン**をクリック
2. 右側に AI 補助パネルが表示

#### Step 2: 自動解析（🔍 解析フェーズ）

- 前後のフレーム画像を自動解析
- シチュエーション理解を生成
- **料金**: 約 0.5-1.5 円

#### Step 3: 状況確認（📝 シチュエーションフェーズ）

- AI 解析結果を確認・編集可能
- 必要に応じて内容を修正
- **この内容で対話を開始**をクリック

#### Step 4: AI 対話（💬 対話フェーズ）

- AI とカジュアルに構図相談
- 短い会話でテンポよく進行
- **料金**: 1 回あたり約 0.1-0.3 円

#### Step 5: 画像生成（🎨 生成フェーズ）

- **🎨 会話内容で画像生成**をクリック
- 対話内容がプロンプトに自動反映
- **Replicate API 設定時**: 実際の AI 画像生成（約 0.3-0.5 円/枚）
- **未設定時**: プレースホルダー画像（無料）

## 💡 効率的な使用のコツ

### 💰 料金を抑える方法

#### 1. 解析回数を最小化

- 同じようなフレームの連続解析を避ける
- 一度解析したら、その結果を活用

#### 2. 対話を簡潔に

- "緊張感"、"明るく"、"ローアングル"など短いキーワードで指示
- 長い説明文より単語の方が効率的

#### 3. バッチ処理

- 複数のフレームをまとめて検討してから実行

### 🎯 効果的な対話例

#### ❌ 非効率な例

```
ユーザー: "この場面をもっとドラマチックで感情的で視覚的にインパクトのある構図にしたいのですが、どのような方法がありますでしょうか？"
```

#### ✅ 効率的な例

```
ユーザー: "ドラマチックに"
AI: "緊張感を出すなら、ローアングルかクローズアップがいいですね！"
ユーザー: "ローアングルで"
AI: "迫力出ますね！キャラクターを見上げる角度で試してみましょう"
```

## 🛠️ トラブルシューティング

### よくある問題と解決方法

#### ❓ "API キーが無効です"

- OpenAI API キーの形式を確認（sk-で始まる）
- API キーに十分な残高があるか確認

#### ❓ "404 エラー"

- モデル名が正しいか確認（gpt-4o 推奨）
- 古いモデル名（gpt-4-vision-preview）は自動更新

#### ❓ "応答が来ない"

- ネットワーク接続を確認
- OpenAI API の利用制限を確認

#### ❓ "料金が予想より高い"

- 長い対話履歴は料金増加の原因
- パネルを閉じて新しいセッションを開始
